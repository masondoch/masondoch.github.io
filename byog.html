BYOG





BYOG is a tile-based, Pac-Man-style game that consists of a character trying to collect a certain number of items that increases each level. The terrain of the levels are generated pseudorandomly using a custom algorithm, and the game also includes a fire path left in the playerâ€™s wake and multiple types of power-ups that provide advantages. It also includes the option to regain lives after death by answering a quiz question. It was made exclusively in Java using a number of libraries, and I singlehandedly designed the logic for the level design and the player options.
